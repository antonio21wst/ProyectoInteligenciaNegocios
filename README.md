# Sistema de Soporte de Decisiones (DSS) para Empresa de Desarrollo de Software


> **Curso:** Inteligencia de Negocios  
> **Institución:** Benemérita Universidad Autónoma de Puebla (BUAP) - Facultad de Ciencias de la Computación 

## Descripción del Proyecto

Este proyecto consiste en el diseño e implementación de un **Sistema de Soporte de Decisiones (DSS)** orientado a una empresa de desarrollo de software. El objetivo principal es transformar los datos operativos en información estratégica mediante un proceso ETL, un Data Warehouse y herramientas analíticas.

El sistema permite automatizar y centralizar la información para mejorar la eficiencia operativa y fomentar una cultura basada en evidencia.

## Objetivos

* **Automatización:** Implementación de un proceso ETL (Extracción, Transformación y Carga).
* **Análisis Multidimensional:** Creación de un cubo OLAP para análisis flexibles.
* **Estrategia:** Integración de un *Balanced Scorecard* con OKRs definidos.
* **Predicción:** Implementación de un modelo predictivo basado en la distribución de Rayleigh para estimación de defectos.
